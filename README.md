# Online Library System

An online library management application built using React, Vite, React Router, Redux Toolkit, and Tailwind CSS.

## GitHub Repository

https://github.com/mayankrawat45/Online-library-Management

## Features

- Home page with categories and popular books
- Browse books by category
- Search books by title or author
- Dynamic book details page
- Add new books using Redux
- Form validation
- Responsive design
- Custom 404 page

## Tech Stack

- React
- Vite
- React Router DOM
- Redux Toolkit
- React Redux
- Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/mayankrawat45/Online-library-Management.git
```

Go to the project folder:

```bash
cd online-library-system
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
